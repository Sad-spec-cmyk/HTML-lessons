CSS відповідає за зовнішній вигляд HTML-сторінки: кольори, розміри, шрифти, відступи, розташування блоків та адаптивність.

HTML + CSS

HTML створює структуру, а CSS задає вигляд цієї структури.

Три способи підключення

CSS можна писати у файлі, в тегу <style> або в атрибуті style.

Каскад

Якщо кілька правил впливають на один елемент, браузер обирає правило за специфічністю і порядком.

Адаптивність

CSS дозволяє робити сторінки зручними для ПК, планшетів і телефонів.

body {
  font-family: Arial, sans-serif;
  background: #f6f7f8;
  color: #222222;
}

CSS-правило складається із селектора, фігурних дужок, властивостей і значень.

селектор {
  властивість: значення;
  інша-властивість: інше-значення;
}
.card {
  background: white;
  border: 1px solid #dddddd;
  padding: 16px;
  border-radius: 8px;
}

Показано всі селектори та властивості.

Селектори вказують, до яких HTML-елементів застосувати CSS-правила.

Селектори за цим запитом не знайдено.

Кожен елемент можна уявити як прямокутник: вміст, внутрішній відступ, рамка і зовнішній відступ.

content

Основний вміст елемента: текст, зображення або інші елементи.

padding

Внутрішній відступ між вмістом і рамкою.

border

Рамка навколо padding і content.

margin

Зовнішній відступ між елементом та сусідніми елементами.

.box {
  width: 300px;
  padding: 20px;
  border: 1px solid #cccccc;
  margin: 16px;
  box-sizing: border-box;
}
